HelloWorld 第一个Tornado应用

2013年3月4日
  1. 建立工作目录
    D:\python\workspace\tornado
  2. 使用Virtualenv建立虚拟环境并激活环境
    初始化环境，进入命令行，执行
    D:\Python\workspace\tornado>virtualenv env
    激活环境，执行
    D:\Python\workspace\tornado>env\Scripts\activate
  3. 安装Tornado 使用pip进行安装，进入命令行激活环境，执行
    (env) D:\Python\workspace\tornado>pip install tornado
  4. 编写HelloWorld.py 
    import tornado.ioloop
import tornado.web

class MainHandler(tornado.web.RequestHandler):
    def get(self):
        self.write("Hello, world")

application = tornado.web.Application([
    (r"/", MainHandler),
])

if __name__ == "__main__":
    application.listen(8888)
    tornado.ioloop.IOLoop.instance().start()
  5. 运行HelloWorld.py
    (env) D:\Python\workspace\tornado>python HelloWorld.py
  6. 测试
    在浏览器中输入http://localhost:8888
好久没有听歌了 Windows 7 安装64位 Python 2.7.3 开发环境

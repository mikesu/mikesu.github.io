Windows 7 安装64位 Python 2.7.3 开发环境

2013年3月4日
  1. 安装Python 2.7.3
    下载http://python.org/ftp/python/2.7.3/python-2.7.3.amd64.msi
    安装到 D:\Python\python27 目录下面
    设置 d:\Python\python27 和 D:\Python\Python27\Scripts 到环境变量Path中
  2. 安装Setuptools
    下载http://peak.telecommunity.com/dist/ez_setup.py到 D:\Python
    安装，进入命令行
    d:\Python>python ez_setup.py
  3. 安装 pip
    下载https://raw.github.com/pypa/pip/master/contrib/get-pip.py到 D:\Python 安装 ，进入命令行
    d:\Python>python get-pip.py
  4. 安装 virtualenv 由于已经安装了pip，我们就可以直接在命令行下用pip install virtualenv进行安装了。 安装，进入命令行
    d:\Python>pip install virtualenv
